# Python Factory

Literalmente uma fábrica de códigos em Python criada para auxiliar implementações, automações e até treinamento de modelos de Machine Learning. Neste repositório será possível encontrar funções e classes para:

- ![#DD3867](https://placehold.it/15/DD3867/000000?text=+) `File System`
  - Funções para auxiliar o manuseio de arquivos gerais em sistemas operacionais, como a validação da presença de determinado arquivo em determinado diretório, validação de atualização de um arquivo, cópia de um arquivo de um diretório origem para um diretório destino, entre outros.
